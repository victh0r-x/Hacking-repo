## Bypass File Upload
_____
- `.php` — la estándar y universal.
- `.phpar`
- `.phtml` — variante histórica muy usada en hosting compartido.
- `.php3` — heredada de PHP 3; aún aceptada en algunos entornos.
- `.php4` — variante de PHP4 (rara hoy, pero aparece).
- `.php5` — para instalaciones con PHP5.
- `.php7` / `.php8` — usadas en hosts que diferencian versión por extensión.
- `.phps` — en Apache muestra **el código fuente** con coloreado (si está habilitado), no ejecuta.
- `.inc` / `.php.inc` — ficheros de inclusión que algunos servidores configuran para parsear como PHP (peligroso si se sirven directamente).
- `.pht` / `.phtml5` — otras variantes que pueden mapearse a PHP en sistemas legacy.
